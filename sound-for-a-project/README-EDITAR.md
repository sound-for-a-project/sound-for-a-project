# Sound for a Project — guía rápida de edición

## Foto principal

La foto principal se carga desde:

```text
images/studio-hero.png
```

Para cambiarla en el futuro, guarda la nueva imagen con ese mismo nombre dentro de la carpeta `images/`.

## Frase bajo la foto

Está en `index.html`. Busca:

```text
EDIT: Hero image caption
```

Texto actual:

```text
A personal mixing environment built around precision, detail and musical intent.
```

## Enlaces pendientes

En `index.html`, busca estos comentarios:

```text
EDIT: YouTube link
EDIT: SoundCloud link
EDIT: KVR link
```

Cuando tengas los enlaces definitivos, cambia sólo `href="#"` por la URL real.

## Redes sociales

En `index.html`, busca:

```text
EDIT: Social media links
```

Ahí están Instagram, TikTok y Facebook.

## Email

En `index.html`, busca:

```text
EDIT: Contact email
```

Recuerda cambiar tanto el `mailto:` como el texto visible si algún día cambia el email.
